## Removed/Deprecated Keys

The following keys have been removed and should not be used:
- ABACUA_API_TOKEN (use ABACUS_API_KEY)
- GEMINI_API_KEY_OB (use GEMINI_API_KEY)
- GITHUB_APP_GENIE_KEY (use GITHUB_PRIVATE_KEY)
- GROK_API_KEY (renamed to GROQ_API_KEY)
- GROK_PROJECT_ID (renamed to GROQ_PROJECT_ID)
- DALLE_ORG_ID (use OPENAI_ORG_ID)
- DALLE_PROJ_ID (use OPENAI_PROJ_ID)
